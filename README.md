# DealBOT - Website Corporativo

**A IA que transforma conversas em negócios**

## Descrição

Site corporativo completo para a startup DealBOT, especializada em automação de atendimento via chatbot integrado com IA. O site apresenta todos os produtos, serviços, análise de mercado e informações relevantes da empresa.

## Estrutura do Projeto

```
dealbot-website/
├── index.html          # Página principal
├── css/
│   └── styles.css      # Estilos CSS completos
├── js/
│   └── script.js       # JavaScript interativo
├── images/
│   └── logo.png        # Logo da DealBOT
├── assets/             # Pasta para recursos adicionais
└── README.md           # Este arquivo
```

## Características

### Design
- **Paleta de Cores**: Verde (#10B981) e Branco, baseada na identidade visual da marca
- **Responsivo**: Totalmente adaptável para desktop, tablet e mobile
- **Moderno**: Design limpo e profissional com animações suaves

### Seções
1. **Hero Section**: Apresentação principal com bordão e CTAs
2. **Sobre**: Missão, visão e diferenciais da DealBOT
3. **Produtos**: Três planos (Basic, Pro, Enterprise) com preços e features
4. **Mercado**: Análise de mercado em Portugal, Brasil e CPLP/PALOP
5. **Benefícios**: 8 benefícios principais com ícones
6. **Competitividade**: Tabela comparativa com concorrentes
7. **Roadmap**: Timeline de expansão 2026-2030
8. **Contato**: Formulário funcional e informações de contato
9. **Footer**: Links úteis e informações corporativas

### Funcionalidades JavaScript
- Menu mobile responsivo
- Scroll suave entre seções
- Animações ao scroll (Intersection Observer)
- Validação de formulário
- Contador animado para estatísticas
- Barra de progresso de scroll
- Efeitos hover interativos

## Como Usar

### Opção 1: Abrir Localmente
1. Extraia o arquivo ZIP
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. Navegue pelo site

### Opção 2: Hospedar Online
1. Faça upload de todos os arquivos para seu servidor web
2. Configure o domínio (ex: www.dealbot.pt)
3. O site estará disponível online

### Serviços de Hospedagem Recomendados
- **Netlify** (gratuito): Arraste a pasta e publique
- **Vercel** (gratuito): Deploy automático
- **GitHub Pages** (gratuito): Hospedagem via repositório
- **Hostinger, SiteGround**: Hospedagem tradicional

## Personalização

### Alterar Cores
Edite as variáveis CSS em `css/styles.css`:
```css
:root {
    --primary-green: #10B981;
    --dark-green: #059669;
    /* ... outras cores */
}
```

### Alterar Conteúdo
Edite diretamente o arquivo `index.html` com qualquer editor de texto.

### Adicionar Funcionalidades
Adicione scripts personalizados em `js/script.js`.

## Formulário de Contato

O formulário atualmente exibe uma mensagem de confirmação. Para conectar a um backend real:

1. Configure um serviço como **Formspree**, **EmailJS** ou **Netlify Forms**
2. Atualize o manipulador de eventos em `js/script.js`
3. Ou integre com API própria
